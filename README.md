# AI Agent Automation in n8n

## Objective
The objective of this project is to automate business workflows using AI-powered agents in n8n for customer engagement, marketing automation, review handling, content generation, and campaign management.

---

## Technologies Used
- n8n
- AI Agents
- Groq API
- JSON Workflows
- Prompt Engineering
- HTTP Requests
- Workflow Automation

---

## Features
- Customer Review Automation
- Festival Campaign Content Generation
- Food Photography Caption & Social Post Generator
- Loyalty Engagement Automation
- Menu Promotion Content Generator
- AI-Based Marketing Content Creation
- Automated Workflow Orchestration

---

## Project Workflows

### 1. Restaurant Review
Automates customer review analysis and generates intelligent responses using AI.

### 2. Festival Campaign Agent
Creates promotional content and campaign ideas for festivals and special events.

### 3. AI food Caption
Generates social media captions and promotional posts for food photography and restaurant marketing.
